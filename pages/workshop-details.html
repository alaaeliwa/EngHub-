Workshop Description

Learn the fundamentals of React, the world's most popular JavaScript library for building user interfaces. This workshop is designed for students who want to transition from traditional HTML/JS to modern component-based frontend development.

We will build a real-world project together, focusing on best practices and modern React patterns (Hooks, Functional Components, and State Management).

What you will learn

React Basics & JSX
Components & Props
State & UseState Hook
Handling Events
Project Structure
Modern UI Patterns

Requirements

  • Basic knowledge of HTML & CSS
  • Familiarity with JavaScript fundamentals (ES6+)
  • Laptop with Node.js installed
